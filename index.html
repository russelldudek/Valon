Why the role exists now

A founding function has to carry four contradictions at once.

The first engagements are signed. The strategist is embedded without a project manager or account executive, expected to build independently, manage change, and create the playbook future hires will inherit.

01

Speed vs. auditability

Ship fast enough to change the business while preserving evidence, compliance, human authority, and recovery.

02

Client specificity vs. platform leverage

Go deep enough to solve the local operation without producing a one-off system that teaches ValonOS nothing.

03

Executive mandate vs. operator trust

Translate ambition into work people can understand, test, own, and improve without pretending fear is resistance.

04

Founder autonomy vs. repeatable method

Act with judgment in the room, then capture the decisions so the next engagement starts smarter.

First 90 days

Earn the right to standardize by shipping one complete engagement.

DAYS 1–15

Enter and map

  • Shadow the signed engagement.
  • Map work, authority, data, and fear.
  • Define the outcome and evidence contract.
DAYS 16–30

Build the first workflow

  • Select one high-value, bounded use case.
  • Prototype the agent or workflow directly.
  • Define human review and recovery.
DAYS 31–60

Operate to proof

  • Run with real operators.
  • Measure quality, exceptions, trust, and economics.
  • Iterate until the outcome exists.
DAYS 61–90

Return and repeat

  • Extract the reusable primitive.
  • Install the engagement playbook.
  • Define quality criteria for the next strategist.
