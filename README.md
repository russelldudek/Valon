# Independent Candidate Vision - Valon

This repository publishes Russell Dudek's independent candidate vision for Valon's **Applied AI Strategist, New Ventures** role.

The campaign is not affiliated with or endorsed by Valon Technologies, Inc.

## Campaign thesis

**Make AI transformation survive contact with the work.**

The role-specific operating model is the **Field-to-Platform Buildline**: enter the live operation, bind outcome to authority, build the smallest real workflow, operate to proof, and return the reusable primitive to ValonOS.

## Artifact set

- `index.html` - interactive candidate vision
- `resume.html` and `docs/Russell-Dudek-Valon-Resume.pdf` - exactly two-page resume
- `cover-letter.html` and `docs/Russell-Dudek-Valon-Cover-Letter.pdf` - exactly one-page cover letter
- `interview-brief.html` and `docs/Russell-Dudek-Valon-Interview-Brief.pdf` - one-page interview thesis brief
- `entry-plan.html` and `docs/Russell-Dudek-Valon-90-Day-Plan.pdf` - one-page 90-day entry plan
- `field-build-brief.html` and `docs/Russell-Dudek-Valon-Field-Build-Brief.pdf` - one-page signature operating brief

## Source and brand notes

The role posting and official Valon company, platform, about, and careers pages informed the role and company analysis. The official wordmark was visible but an original downloadable brand file could not be retrieved through the connected environment, so the campaign uses a non-official typeset company identifier rather than tracing or recreating the mark.
